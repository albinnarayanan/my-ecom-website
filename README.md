Project Description:
This project is a full-stack e-commerce web application designed to deliver a seamless shopping experience with robust backend architecture and a modern, responsive frontend.

Key Features:
- Product browsing: Users can explore a catalog of items with intuitive navigation.
- Order placement: Streamlined checkout flow with cart management and order tracking.
- Account management: Secure user profiles with role-based access for customers, sellers, and admins.
- Responsive UI: Optimized for usability and accessibility across desktop and mobile devices.
- Stripe integration: Secure online payments with PCI-compliant transactions.
- JWT authentication: Strengthened security with token-based login and role-specific dashboards.

Tech Stack:
- Frontend: React, JavaScript
- Backend: Java, Spring Boot
- Database: MySQL
- API Testing: Postman
